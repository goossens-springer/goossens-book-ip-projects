Some examples

  • Recommended usage: Encode a WAVE-file myfile.wav to a bitrate of 128000 bits/s, writing to a plain bitstream myfile.mp3 and using high quality mode:

    • lame -h myfile.wav myfile.mp3

  • Encode a plain PCM stream (2-channel, 44.1 kHz) to a plain 56 kBit/s Layer-3 stream, using the encoder as filter

    • lame -r -s 44.1 -b 56 - -
